# PriceSeer: Evaluating LLMs in Real-Time Stock Prediction

- **arXiv**: [2601.06088](https://arxiv.org/abs/2601.06088)
- **日期**: 2026-01-06
- **子领域**: 时序预测 / LLM 评估

## 核心问题
LLM 能否预测股票价格？目前缺乏动态、无污染的标准基准。

## 方法
**PriceSeer 基准**:
- 110 只多行业股票
- 动态数据流 (非静态快照, 避免数据泄露)
- 多窗口评估 (1/5/10/20天)
- 数据污染抵抗力测试

## 关键发现
1. LLM 短期预测略优于随机 (52-55%)
2. 长期预测 (>20天) 接近随机
3. 容易被虚假信息误导
4. 校准度差 (过度自信)
