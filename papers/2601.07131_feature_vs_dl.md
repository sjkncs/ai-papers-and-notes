# Feature Engineering Beats Deep Learning in Flow Prediction

- **arXiv**: [2601.07131](https://arxiv.org/abs/2601.07131)
- **日期**: 2026-01-07
- **子领域**: 因子工程 / 资金流预测

## 核心问题
在投资者资金流预测中, 复杂深度学习是否真的比传统方法更好？

## 方法
- 对比: XGBoost + 特征工程 vs MLP/Transformer + 原始特征
- 特征类别: 历史流量、市场表现、日历效应、基金属性、交叉特征
- 数据: 100只模拟基金, 10年日频

## 关键发现
1. **XGBoost + 特征工程 > MLP + 原始特征** (R² 和方向准确率均优)
2. 交叉特征 (市场×属性) 是关键
3. 模型复杂度不是答案, 特征质量决定上限

## 量化启示
- **先做好特征, 再考虑模型**
- 对 alpha 因子挖掘: domain knowledge > model architecture
- 简单模型 + 好特征 = 更可解释、更稳定
